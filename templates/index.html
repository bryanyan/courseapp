CMU Courses

CMU Courses provides information about courses at Carnegie Mellon including descriptions, pre/coreqs, and course evaluations.

CMU Courses was built to showcase the ScottyLabs Course-API . This application is built on Tornado, a Python web framework and asynchronous networking library as well as Bootstrap. The source code for this application can be found here.

If there are any bugs or suggestions you would like to make, please contact bryanyan@andrew.cmu.edu
